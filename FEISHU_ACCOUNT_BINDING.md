# 飞书账号关联指南

本文档详细说明如何将飞书账号与 Info-Butler 网页版账号进行关联。

---

## 关联方式

Info-Butler 提供 **3 种账号关联方式**：

### 方式 1：自动关联（推荐）✅

**适用场景**：您在网页版注册时使用的邮箱与飞书邮箱一致。

**操作步骤**：无需任何操作！
1. 首次在飞书中@机器人发送消息
2. 系统自动检测飞书邮箱
3. 如果邮箱匹配，自动完成关联
4. 立即可用所有功能

**示例**：
- 网页版注册邮箱：`zhangsan@company.com`
- 飞书邮箱：`zhangsan@company.com`
- 系统自动识别并关联 ✅

---

### 方式 2：绑定码关联 ✅

**适用场景**：
- 已有网页版账号
- 飞书邮箱与注册邮箱不一致
- 希望手动控制关联过程

#### 步骤 1：在网页端生成绑定码

1. 登录 Info-Butler 网页版
2. 进入【个人设置】页面
3. 点击【绑定飞书账号】
4. 系统生成 6 位绑定码，例如：`A1B2C3`
5. 绑定码 **5 分钟内有效**

#### 步骤 2：在飞书中完成绑定

在飞书中@机器人发送：

```
绑定 A1B2C3
```

（替换为您实际的绑定码）

#### 步骤 3：确认绑定成功

机器人回复：

```
✅ 绑定成功！

您的飞书账号已成功关联到 Info-Butler 账号。
现在可以正常使用所有功能了。
```

---

### 方式 3：自动创建新账号 ✅

**适用场景**：新用户，没有网页版账号。

**操作步骤**：
1. 在飞书中@机器人发送任意消息
2. 系统自动为您创建新账号
3. 账号信息：
   - 用户名：飞书昵称
   - 邮箱：飞书邮箱（或临时邮箱）
   - 密码：随机生成（可后续在网页端修改）
4. 立即可用所有功能

**注意**：
- 如需在网页端登录，请先通过【忘记密码】功能设置密码
- 或联系管理员获取账号信息

## 常见问题

### Q1: 如何查看是否已关联？

**方式 1：在网页端查看**
1. 登录网页版
2. 进入【个人设置】
3. 查看"飞书账号绑定状态"

**方式 2：在飞书中查看**
1. @机器人发送：`帮助`
2. 如果已关联，系统会显示您的用户信息

### Q2: 邮箱不一致怎么办？

如果您的飞书邮箱与注册邮箱不一致，请使用 **方式 2：绑定码关联**。

### Q3: 一个飞书账号可以绑定多个网页账号吗？

❌ 不可以。一个飞书账号只能绑定一个 Info-Butler 账号。

### Q4: 一个网页账号可以绑定多个飞书账号吗？

❌ 不可以。一个 Info-Butler 账号只能绑定一个飞书账号。

### Q5: 如何解除绑定？

1. 登录网页版
2. 进入【个人设置】
3. 点击【解绑飞书账号】
4. 确认解绑

**注意**：解绑后，飞书中的操作将创建新账号。

### Q6: 绑定码过期了怎么办？

绑定码有效期为 **5 分钟**。如果过期：
1. 重新在网页端生成绑定码
2. 在飞书中发送新的绑定码

### Q7: 绑定失败怎么办？

**可能原因及解决方案**：

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| "无效的绑定码" | 绑定码输入错误 | 检查绑定码是否正确 |
| "绑定码已过期" | 超过5分钟 | 重新生成绑定码 |
| "绑定码已使用" | 绑定码已用过 | 重新生成绑定码 |
| "该飞书账号已被其他用户绑定" | 飞书账号已绑定其他账号 | 先解绑原账号 |
| "已绑定飞书账号" | 网页账号已绑定其他飞书 | 先解绑原飞书账号 |

## 技术实现

### 自动关联逻辑

```
1. 用户在飞书中@机器人发送消息
2. 系统获取飞书 open_id
3. 查找是否有已绑定的用户
   - 有 → 直接返回用户
   - 无 → 继续步骤 4
4. 获取飞书用户信息（包括邮箱）
5. 通过邮箱查找现有用户
   - 找到 → 绑定 open_id 并返回用户
   - 未找到 → 创建新用户
```

### 绑定码流程

```
网页端：
1. 用户点击"生成绑定码"
2. 系统生成 6 位随机码（如 A1B2C3）
3. 存储绑定码 + user_id + 过期时间
4. 显示绑定码给用户

飞书端：
1. 用户发送"绑定 A1B2C3"
2. 机器人解析绑定码
3. 调用绑定 API
4. 验证绑定码有效性
5. 更新用户的 feishu_open_id 字段
6. 返回绑定成功消息
```

## 故障排查

### 问题：绑定后看不到历史数据

**原因**：绑定了错误的账号

**解决方案**：
1. 解绑当前飞书账号
2. 确认网页版账号信息
3. 重新绑定正确的账号

### 问题：飞书中创建的待办在网页端看不到

**可能原因**：
1. 账号未正确关联
2. 数据同步延迟

**解决方案**：
1. 检查绑定状态
2. 刷新网页端
3. 重新发送消息测试

### 问题：自动创建了新账号，但我想绑定现有账号

**解决方案**：
1. 在网页端登录原有账号
2. 生成绑定码
3. 在飞书中发送绑定码
4. 系统会提示"该飞书账号已被绑定"
5. 联系管理员处理（需要手动调整数据库）
